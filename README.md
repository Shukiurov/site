# Ali Shukiurov — Portfolio

## Структура файлов

```
ali-portfolio/
├── index.html       — разметка (HTML)
├── style.css        — все стили
├── script.js        — логика (переводы, навигация, анимации)
├── photo.jpg        — СЮДА положи своё фото (см. ниже)
└── README.md        — этот файл
```

## Как добавить своё фото

В файле `index.html` найди строку:
```html
src="https://images.unsplash.com/photo-..."
```
Замени на:
```html
src="photo.jpg"
```
И положи своё фото рядом с `index.html` под именем `photo.jpg`.

## Как редактировать текст

- Открой `index.html` в браузере
- Нажми кнопку **✏ Edit** в правом нижнем углу
- Редактируй текст прямо на странице

Или открой `script.js` и редактируй переводы в объекте `T` (строки с `ru:`, `en:`, `nl:`).

## Контакты (как изменить)

В `script.js` найди объект `T` и измени значения.
В `index.html` найди блок `id="contCell"` и измени ссылки.

## Запуск

Просто открой `index.html` в браузере.
Для размещения онлайн — загрузи **все файлы** на хостинг (Netlify, GitHub Pages и т.д.).
